Thalassa Brand Palette (v1) — locked 2026-05-17

These hexes are the canonical brand colours, used ONLY on conversion-
moment surfaces: SignInScreen, UpgradeModal, favicons, app icon,
splash, OG images. The everyday UI surfaces (Glass / Charts / Plan /
Log / Vessel and every sub-page they open into) use a different
palette — sky-blue accents on slate, defined in `theme.ts`.

Mirror in code at `theme.ts`:
    import { BRAND } from './theme';

Cleaned `full-lockup-dark.svg` uses three of these:
  - #FFFFFF  White            (wordmark + compass body)
  - #5EEAD4  Sea-foam teal    (wave swirl + accents)
  - (background transparent, sits on slate-950 #020617 from theme)

Cleaned `full-lockup-light.svg` uses three of these:
  - #FFFFFF  White            (background)
  - #0F172A  Primary navy     (wordmark + compass body)
  - #5EEAD4  Sea-foam teal    (wave swirl + accents)

DO NOT use these colours on everyday UI surfaces. If you find
yourself reaching for one of these on a Vessel sub-page or a Log
entry, either (a) you're on a conversion moment that needs a design
call, or (b) you should be using the UI palette in theme.ts. The
two-palette discipline is what keeps "Thalassa is talking to you"
moments from getting drowned out by routine app chrome.
